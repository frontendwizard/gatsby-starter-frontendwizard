# gatsby-starter-frontendwizard

A Gatsby starter for my projects. This a hello-world with some extra features:

- Tailwind.css
- Jest
- MDX with images
- Typescript
- Storybook
- Prettier
- Eslint with a few plugins:
  - @typescript-eslint
  - prettier
  - react
  - react-hooks
  - import
  - mdx
- commitlint with conventional config
- Cypress
